## About me

I'm Nazrin — a backend engineer from Malaysia who ended up, somewhat by accident, becoming
the person who gets called when a payment goes missing. I build the plumbing behind
transactions: the APIs, the queues, the databases, the parts nobody notices until they
break.

Right now I'm at **Fiuu** (formerly Razer Merchant Services), building the Golang services
and Laravel APIs that keep credit/debit, BNPL, recurring billing, and DuitNow QR payments
flowing for merchants like Shopee, TikTok, and Zus Coffee. Before that: government tender
systems handling sensitive data at scale, an accounts-receivable platform used by 90% of
Malaysia's debt collection agencies, and — going further back — a computer vision project
that taught a model to tell when a palm oil bunch was ripe. Full story is on the site.

A few things that are true about how I work:
- I'd rather a system be boring and correct than clever and fragile.
- I like backend work because the constraints are real — throughput, integrity, uptime —
  not aesthetic opinions.
- I still enjoy the occasional frontend detour, which is partly why this portfolio exists.

## What's in this repo

This is the source for my personal site — one `index.html`, no build step, no framework.
That's deliberate: a portfolio for a backend engineer should still be something a backend
engineer could explain end-to-end without a bundler in the way.

| | |
|---|---|
| **Stack** | HTML, CSS, vanilla JS, [Three.js](https://threejs.org/) for the background |
| **Hosting** | GitHub Pages, custom domain via `CNAME` |
| **Design** | Black & gold, monospace accents — a "deep space terminal" aesthetic |
| **Notable bits** | Scroll-reveal animations, a presentation/slideshow mode, a floating contact widget, count-up stats, a typewriter hero line |

## Running it locally

There's nothing to install:

```bash
git clone https://github.com/NazrinAfzal1997/portfolio-nazrin-web.git
cd portfolio-nazrin-web
python3 -m http.server 8080   # or just open index.html directly
```
